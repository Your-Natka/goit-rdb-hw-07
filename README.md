# Домашнє завдання Тема 7: Додаткові вбудовані SQL функції. Робота з часом

## Опис
У цьому завданні ми практикуємося працювати з часовими функціями та JSON-об'єктами у MySQL.

- Витягаємо рік, місяць та день із дати.
- Додаємо інтервал до дати.
- Переводимо дату у timestamp.
- Рахуємо кількість рядків у певному діапазоні дат.
- Створюємо JSON-об'єкт із атрибутів таблиці.

## Виконання
1. Підключаємося до вашої MySQL бази даних.
2. Виконуємо скрипт `homework07.sql`.
3. Робимо скриншоти результатів виконання кожного запиту.
4. Додаємо скриншоти та файл `homework07.sql` у ваш репозиторій.
